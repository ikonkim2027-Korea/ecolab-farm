🧪 Experiment Flow

1. Hypothesis → "More CO₂ = faster growth" 2. Setup → Control + Experimental group 3. Run → Side-by-side simulation 4. Observe → Auto-generated graphs 5. Conclude → Report template provided

Students learn the scientific method by doing it — not memorizing steps from a textbook.

DIFFICULTY SYSTEM

Same World, Different Depth

Difficulty = number of visible variables. The simulation underneath is identical.

🌱 Easy Grades 4–5

Manage: Water, Sunlight Events: Rare, simple Speed: Up to 16x Neglect: 6hr grace period Science: Simple gauges Lab: Guided experiments

🌿 Normal Grades 5–7

Manage: + NPK, Pests Events: Regular Speed: Up to 8x Neglect: 3hr grace period Science: Numbers + graphs Lab: Semi-guided

🌳 Hard Grades 7–8

Manage: + pH, CO₂, Temp, Ecosystem, Economics Events: Frequent + compound Speed: Up to 4x Neglect: 1hr grace period Science: Molecular + formulas Lab: Full freedom

Switch anytime. Progress is never lost.

PLATFORM

Pixel Art. Web. Mobile. ~10MB.

🎨 Art Style
